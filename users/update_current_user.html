User

A User represents a person or automated actor on GitHub. They may own and contribute to repositories and belong to zero or more organizations. Users have email addresses and ssh keys, as well as other relevant profile data.

Representation

GitHub API v3

application/vnd.github.v3+json

Attributes

login
  • String
The login name for the user
name
  • String
The users human meatspace name
last_login
  • ISO8601 Date
Timestamp when the user last logged in
organizations_href
  • URL
Link to the organizations the user is a public member of
emails_href
  • URL
Link to the public emails registered to the user
ssh_keys_href
  • URL
Link to the public ssh keys for the user
