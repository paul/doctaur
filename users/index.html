User

A User represents a person or automated actor on GitHub. They may own and contribute to repositories and belong to zero or more organizations. Users have email addresses and ssh keys, as well as other relevant profile data.

Representation

Attributes

login
  • string
The login name for the user
name
  • string
The user's meatspace name
last_login
  • iso8601
Date and time of the user's last login to github.com

Links

organizations_href
  • organization
[Organizations](#organization) of which this user is a public member
emails_href
  • email
The public [emails](#mail) registered to this user
ssh_keys
  • key
Public [SSH keys](#ssh_key) for this user
